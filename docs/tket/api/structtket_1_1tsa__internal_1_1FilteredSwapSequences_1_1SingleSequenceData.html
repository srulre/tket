This is how we access the fixed data in the large const static global table.

This constructor looks up the shortest sequence of swaps enacting the given permutation, and fills the entries.

Parameters
permutation_hashThe hash of the desired permutation of {0,1,2,...,5}, as used to look up results in the table (after relabelling). See CanonicalRelabelling for explanation.
edges_bitsetThe collection of edges on {0,1,2,...,5} which actually exist in the graph (i.e., the swaps which are allowed).
max_number_of_swapsDo not return any solutions with more swaps than this: useful speedup to allow early termination.
